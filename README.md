# graphql-usermanager

> A User Manager build w/ GraphQL Backend

## Includes

- Webpack 3
- Babel (ES6, babel-preset-env)
- Hot-Reload
- ESLint w/ Standard
- Express
- graphql-tag (Support for .graphql files)
- Apollo

## Usage

```bash
git clone https://github.com/chryb/graphql-usermanager

cd graphql-apollo-boilerplate

# make it to your own
rm -rf .git

npm install

# hot reload
npm run dev

# build
npm run build

# build & start
npm start

# test
npm run unit
```
